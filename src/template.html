Background Generator

Background Generator is an interactive webpage meant first of all for the developers. From firsthand experience, I realized how tempting picking a gradient background for a component might be. Background generator offers an easy way to choose a perfect gradient for your background. Any color combination a user picks is immediately displayed, so that one instantly sees the outcome. When a perfect combination is chosen, all what is left to do is copy and paste the line of CSS that will add the exact gradient background to one’s app.

Robofriends is a first React app ever developed by me. EasySchool

EasySchool is a React Application developed for a real life client (a teacher at a school). The purpose of the application was helping the teacher to keep track of students’ assignments. The development of the project involved client’s feedback on various stages, troubleshooting, deployment and production of a documentation which included the rationale for development, solution overview, justification of techniques, video demonstration and evaluation of the project. EasySchool was the most difficult project I have developed especially in terms of logical manipulation of state in React components. This project proved to me that as the applications get bigger and bigger, special state manipulation libraries, such as Redux, become extremely helpful.

SmartBrain

SmartBrain is a project developed using React as well. When opened, the home screen offers the user to log in or register. Then, the user can submit an Url of a picture and SmartBrain will display the blue box around the face of one or multiple people on the image. SmartBrain is not a face detection application by itself, it makes an Api call to Clarifai – very popular collection of different AI models. Besides the front end, I have also developed the backend – a RESTful Api using Node.js and Express.js. In attempts to store users’ information securely, I have decided to store hashes of users’ passwords in a separate table inside a relational database.
